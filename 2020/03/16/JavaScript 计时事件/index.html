JavaScript 计时事件

发表于 2020-03-16 | 分类于 JavaScript

Javascript计时事件
通过使用JavaScript，设计一个时间间隔之后来执行代码,而不是在函数被调用后立即执行，这就时计时事件
在JavaScript中使用计时事件，两个关键方法是：

  • setInterval()间隔指定的毫秒数不停地执行指定地代码
  • setTimeout()在指定地毫秒数后执行指定代码
    setInterval()和setTimeout()是HTML DOM Window对象的两个方法

setInterval()方法
setInterval()间隔指定的毫秒数不停地执行指定地代码
语法

window.setInterval()方法可以不适用window前缀,直接使用函数setInterval()
setInterval()第一个参数是函数(function)
第二个参数间隔的毫秒数
1000毫秒是1秒

显示当前时间

clearInterval()
clearInterval()方法用于停止setInterval()方法执行函数代码
语法

window.clearInterval()方法可以不使用前缀window，直接使用函数clearInterval()
要使用clearInterval()方法，在创建计时方法是必须使用全局变量

然后可以使用clearInterval()方法来停止执行

setTimeout()方法
语法

setTimeout()方法会返回某个值,在上面的语句中，值被存储在名为myVar变量中，如果要取消这个setTimeout(),可以使用这个变量名来指定它
setTimeout()的第一个参数是函数JavaScript语句的字符串
第二个参数指示从当前起多少毫秒后执行第一个参数

clearTimeout()方法
clearTimeout()方法用于停止执行setTimeout()方法的函数代码
语法

window.clearTimeout()方法可以不使用window前缀
要使用clearTimeout()方法，必须在创建超时方法中(setTimeout)使用全局变量

如果函数还未被执行，可以使用clearTimeout()方法来停止执行函数代码

jQuery 事件

发表于 2020-03-17 | 分类于 jQuery

jQuery事件方法语法
在jQuery中，大多数DOM事件都有一个等效的jQuery方法
页面中指定一个点击事件

下一步是定义什么时间触发事件，可以通过一个事件函数实现

常见的jQuery事件方法
$(document).ready()方法允许在文档完全加载完成后执行函数
click()方法是按钮点击事件被触发时会调用一个函数
该函数在用户点击HTML元素时执行
当点击事件在某个

元素上触发时,隐藏当前

元素

dblclick()
当双击元素时,会发生dblclick事件
dblclick()方法触发dblclick事件，或规定当发生dblclick事件时运行的函数

mouseenter()
当鼠标指针穿过元素时,会发生mouseenter事件
mouseenter()方法触发mouseenter事件，或规定当发生mouseenter事件时运行的函数

mouseleave()
当鼠标指针穿过元素时,会发生mouseleave事件
mouseleave()方法触发mouseleave事件，或规定当发生mouseleave事件时运行的函数

mousedown()
当鼠标指针移动到元素上方并且按下鼠标按钮时,会发生mousedown事件
mousedown()方法触发mousedown事件，或规定当发生mousedown事件时运行的函数

mouseup()
当鼠标指针移动到元素上方并且点击后松开鼠标按钮时,会发生mouseup事件
mouseup()方法触发mouseup事件，或规定当发生mouseup事件时运行的函数

hover()方法用于模拟国标悬停事件
当鼠标移动到元素上时,会触发指定的第一个函数，当鼠标移出这个元素时,就会触发的第二个函数

focus()
blur()
当元素获取到焦点时,发生focus事件
当元素失去到焦点时,发生blur事件
当通过鼠标点击选中元素或通过tab建定位到元素时，该元素就会获得焦点
focus()方法触发focus事件,或规定当发生focus事件时运行的函数
blur()方法触发blur事件,或规定当发生blur事件时运行的函数

jQuery 语法

发表于 2020-03-17 | 分类于 jQuery

jQuery语法
jQuery语法通过选取HTML元素，并对选取的元素执行操作
基础语法：$(selector).action()

  • 美元符号定义jQuery
  • 选择符(selector)”查询”和”查找”HTML元素
  • jQuery的action()执行对元素的操作
    实例
  • $(this).hide() 隐藏当前元素
  • $(“p”).hide() 隐藏所有

    元素

  • $(“p.test”).hide() 隐藏所有class=”test”的

    元素

  • $(“#test”).hide() 隐藏所有id=”test”的元素
    Jquery使用的语法是XPath与CSS选择器语法的组合

Jquery写法

简洁写法

jQuery入口函数
$(document).ready(function(){
执行代码
})
或者
$(function(){
执行代码
})
JavaScript入口函数
window.onload = function(){
执行代码
}
Jquery入口函数与JavaScript入口函数的区别

  • jQuery的入口函数是在html所有标签(DOM)都加载之后，就回去执行
  • JavaScript的window.onload事件是等到所有内容，包含外部图片之类的文件加载完后，才会执行
